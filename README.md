# SoundCloud Media Feed Tracker

Shows played songs on your soundcloud feed, saves a history with their information, SCTracker.* has export and many other utility functions.

## Functions

You can use this functions from the devconsole:

exportSongs() - exports all played songs to a JSON file.

topArtists(n) - prints the top n most played artists.

topPosters(n) - prints the top n posters loaded in your feed, good to catch spammers.

topTracks(n) - prints the top n most played tracks.

recentTracks(n) - prints the last n played tracks.
